About

เว็บนี้คือเดโม Thairecheckpump แบบ frontend-first

หน้าเว็บชุดนี้สร้างขึ้นจากแนวคิดของ `thaipumpradar.com` แล้วออกแบบใหม่ให้เป็น static site หลายหน้า เน้นให้เห็นทั้งเรดาร์สถานี ฟีดรายงาน คลังภาพ และแดชบอร์ดข้อมูลภายใต้ดีไซน์เดียวกัน

1

เก็บสัญญาณจากผู้ใช้

ฟอร์มหน้า Report ใช้จำลองการรับรายงานสถานี ชนิดน้ำมัน และสถานะล่าสุด จากนั้นเก็บไว้ฝั่ง browser เพื่อให้ flow ทำงานครบโดยไม่ต้องมี backend

2

รวมเป็นมุมมองสด

ข้อมูลชุดเดียวกันถูกนำไปเรนเดอร์ที่หน้าแผนที่ ฟีด คลังภาพ และแดชบอร์ด ทำให้ทุกหน้าเล่าเรื่องเดียวกันและอัปเดตไปพร้อมกัน

3

ขยายเป็น production ได้

ถ้าจะต่อยอดจริง สามารถเปลี่ยน data source จาก array ใน `app.js` ไปเป็น API, map provider และ media upload ได้โดยไม่ต้องรื้อโครงสร้างหน้าใหม่

Scope

สิ่งที่มีในเดโม

หลายหน้า, responsive layout, filter เชื้อเพลิง/สถานะ/แบรนด์, form submission แบบ local และ visual language เดียวกันทั้งระบบ

Data

ข้อจำกัดปัจจุบัน

ข้อมูลสถานีและภาพเป็นข้อมูลจำลอง ไม่ได้เชื่อม API จริง และหน้าแผนที่ใช้ radar canvas-style แทนแผนที่จริงเพื่อให้ใช้งานได้ทันทีแบบไม่พึ่ง dependency ภายนอก

Next
