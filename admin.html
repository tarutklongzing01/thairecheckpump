Admin Access

Workflow

ทางลัดสำหรับงานหลังบ้าน

ดูสถานะด้านบนก่อน จากนั้นนำเข้าข้อมูล ค้นหารายการฝั่งซ้าย แล้วค่อยแก้ไขและบันทึกจากฟอร์มฝั่งขวา

Import Stations Reports
1. Access

ตรวจสิทธิ์ก่อนแก้

เช็กโหมด, สิทธิ์เขียน, และ Firebase health ด้านบนก่อนเริ่มแก้ข้อมูลจริงทุกครั้ง

2. Import

มือถือก็ดึงได้

ถ้าไม่มีไฟล์ JSON ในเครื่องให้ใช้ปุ่มดึงตรงหรือดึงแล้วนำเข้าได้ทันทีจากหน้าหลังบ้าน

3. Review

คิวซ้าย ฟอร์มขวา

รายการค้นหาอยู่ฝั่งซ้าย ส่วนรายละเอียดและปุ่มบันทึกถูกเก็บไว้ฝั่งขวาเพื่อลดการสับสนเวลาใช้งาน

Import

นำเข้าข้อมูลจาก PumpRadar

ใช้ JSON จาก `https://thaipumpradar.com/api/provinces/<province-slug>/stations` แล้วนำเข้าเป็น `stations` ใน Firestore โดยใช้ `station.id` เป็น Document ID ระบบจะสร้าง `fuelStates`, `reportCount`, `updatedAt` และ metadata พื้นฐานให้พร้อมใช้งานกับหน้าแผนที่ทันที

Mobile

ดึงแล้วนำเข้า

เหมาะกับการอัปเดตจากมือถือหรือเวลาที่ไม่มีไฟล์ JSON อยู่ในเครื่อง

JSON

วางหรืออัปโหลดก่อน

เหมาะเวลาต้องการตรวจ payload ก่อน import หรือมีไฟล์รวมหลายจังหวัดอยู่แล้ว

Source

เช็กผลลัพธ์หน้า public

ถ้า public ยังใช้ Static JSON อยู่ การ import รอบนี้จะอัปเดตใน Firestore ก่อน

ดาวน์โหลด JSON
ปุ่มนี้จะเปิด API ของ PumpRadar ในแท็บใหม่ แล้วค่อยบันทึกไฟล์เป็น `.json` ปุ่มดึงตรงและดึงแล้วนำเข้าใช้ได้ทั้งมือถือและคอมผ่าน Netlify function ของเว็บ ถ้าขึ้น 404 แปลว่ายังไม่ได้ deploy ฟังก์ชันขึ้น Netlify หรือไม่ได้เปิดผ่าน netlify dev
กำลังตรวจสอบ source ของสถานี
Stations

เลือกรายการที่ต้องการแก้ไข

Station Queue

ค้นหาจากชื่อ แบรนด์ พื้นที่ หรือ Document ID แล้วเลือกจากคิวด้านล่างเพื่อโหลดฟอร์มแก้ไข

Editor

เลือกสถานีเพื่อแก้ไข

เลือกสถานีจากรายการด้านซ้าย หรือกดสร้างสถานีใหม่

Station Editor

รวมข้อมูลหลักของสถานี พิกัด รูป และสถานะเชื้อเพลิงไว้ในแผงเดียวเพื่อให้ตัดสินใจและบันทึกได้เร็วขึ้น

สถานะน้ำมันแต่ละชนิด
Reports

จัดการรายงานที่ผู้ใช้ส่งมา

Report Queue

ใช้คิวนี้สำหรับตรวจรายงานผิดพลาด เปลี่ยนสถานะ หรือย้ายรายงานไปยังสถานีที่ถูกต้องจากหน้าเดียว

Report Editor

เลือกรายงานเพื่อแก้ไข

เลือกรายงานจากรายการด้านซ้าย ระบบจะบันทึกกลับไปยัง Firestore และ sync สรุปของสถานีให้

Report Editor

แก้ชื่อสถานี ผู้รายงาน สถานะน้ำมัน พิกัด และรูปได้จากตรงนี้ แล้วระบบจะ sync ภาพรวมของสถานีให้อีกชั้น
