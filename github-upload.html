Flow

หน้ารวมสำหรับ data และ deploy

ความปลอดภัย thairecheckpump ใช้ Fine-grained token ที่มีสิทธิ์ Contents: Read and write

ถ้าเปิดหน้า uploader จากเว็บ live แล้วต้องการ push ไฟล์โค้ดใหม่ ให้เลือกโฟลเดอร์โปรเจกต์จากเครื่องก่อน เพื่อให้ browser อ่านไฟล์ที่ยังไม่ถูก deploy ได้
